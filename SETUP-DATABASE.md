# تعليمات إعداد قاعدة البيانات

## ⚠️ خطوة مهمة جداً

لتفعيل نظام إدارة الحقول، يجب تنفيذ السكريبت التالي في Supabase:

### الخطوات:

1. افتح Supabase Dashboard: https://supabase.com/dashboard
2. اختر المشروع الخاص بك
3. اذهب إلى **SQL Editor** من القائمة الجانبية
4. انسخ محتوى ملف `supabase/schema.sql` بالكامل
5. الصقه في SQL Editor
6. اضغط **Run** أو **F5**

### ما سيتم إنشاؤه:

✅ جدول `step_fields` - لتخزين حقول الخطوات القابلة للتخصيص
✅ البيانات الأولية - جميع الحقول الحالية للخطوات الـ 6
✅ RLS Policies - للتحكم في الصلاحيات

### بعد التنفيذ:

- افتح `/dashboard/settings` → "حقول الخطوات"
- ستظهر جميع الحقول الحالية قابلة للتعديل
- يمكنك إضافة/حذف/تعديل الأسئلة بحرية

---

## التحقق من نجاح التثبيت

قم بزيارة: `/api/admin/step-fields?step=1`

يجب أن ترى JSON يحتوي على حقول الخطوة الأولى.
