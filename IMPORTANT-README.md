# ⚠️ تعليمات مهمة جداً - اقرأ أولاً!

## 🔴 خطوة إلزامية قبل الاستخدام

**يجب تنفيذ السكريبت في قاعدة البيانات Supabase أولاً!**

### الخطوات:

1. **افتح Supabase Dashboard:**
   - اذهب إلى: https://supabase.com/dashboard
   - اختر مشروعك

2. **افتح SQL Editor:**
   - من القائمة الجانبية، اختر **SQL Editor**
   - أو اضغط على "New query"

3. **انسخ والصق:**
   - افتح ملف `supabase/schema.sql`
   - انسخ **كل المحتوى**
   - الصقه في SQL Editor

4. **نفّذ السكريبت:**
   - اضغط زر **Run** (أو F5)
   - انتظر حتى يكتمل التنفيذ
   - يجب أن ترى رسالة "Success"

## 🆘 إذا واجهت مشاكل:

1. **لا تظهر الحقول؟**
   - تأكد من تنفيذ `schema.sql` في Supabase
   - تحقق من متغيرات البيئة (.env.local)

2. **خطأ في API؟**
   - تأكد من `NEXT_PUBLIC_SUPABASE_URL`
   - تأكد من `SUPABASE_SERVICE_ROLE_KEY`

3. **الحقول فارغة؟**
   - افحص: `/api/admin/step-fields?step=1`
   - إذا كان فارغاً، أعد تنفيذ السكريبت
