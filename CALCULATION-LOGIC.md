# 📐 منطق الحسابات - مراعي الشمال

## 🧮 حساب العجز/الزيادة

### الصيغة:
```
رصيد أمس + وارد اليوم - المبيعات - الصادر - المخلفات = المفروض يتبقى

المتبقي الفعلي (من الثلاجة) - المفروض يتبقى = العجز أو الزيادة
```

### مثال عملي:

#### حاشي:
```
رصيد أمس:      100 كجم
وارد اليوم:     50 كجم
المبيعات:       80 كجم
الصادر:         10 كجم
المخلفات:       5 كجم
───────────────────────
المفروض يتبقى:  55 كجم (100 + 50 - 80 - 10 - 5)

المتبقي الفعلي: 52 كجم
───────────────────────
العجز:           3 كجم (52 - 55) ⚠️
```

#### حالة الزيادة:
```
المفروض يتبقى:  55 كجم
المتبقي الفعلي: 58 كجم
───────────────────────
زيادة:          3 كجم (58 - 55) ✅
```

## 💰 مطابقة الأموال

### الصيغة:
```
كاش + شبكة + آجل + تحويل بنكي = يجب أن يساوي إجمالي المبيعات
```

### مثال:
```
إجمالي المبيعات:  10,000 ريال

توزيع الأموال:
- كاش:           5,000 ريال
- شبكة:          3,000 ريال
- آجل:           1,500 ريال
- تحويل بنكي:     500 ريال
───────────────────────────
المجموع:         10,000 ريال ✅ مطابق
```

#### حالة عدم التطابق:
```
إجمالي المبيعات:  10,000 ريال
مجموع الأموال:    9,800 ريال
───────────────────────────
عجز الصندوق:      200 ريال ⚠️
```

---

## 🔄 رصيد اليوم السابق

### القواعد:
1. **يُحسب تلقائياً** من التقرير السابق
2. **الكاشير لا يمكنه تعديله** (للقراءة فقط)
3. **إذا لا يوجد تقرير سابق** = 0
4. **يُعرض في بداية الخطوة 5** (المتبقي)

### مصدر البيانات:
```
رصيد أمس الحاشي = المتبقي الفعلي من تقرير أمس (الخطوة 5)
رصيد أمس الغنم  = المتبقي الفعلي من تقرير أمس (الخطوة 5)
رصيد أمس العجل  = المتبقي الفعلي من تقرير أمس (الخطوة 5)
```

---

## 🎨 ألوان التنبيهات

### العجز:
- **🟢 لا يوجد عجز (0)** → أخضر
- **🟡 عجز بسيط (1-5 كجم)** → أصفر
- **🔴 عجز كبير (>5 كجم)** → أحمر

### الأموال:
- **🟢 مطابق تماماً** → أخضر
- **🟡 فرق بسيط (<100 ريال)** → أصفر
- **🔴 فرق كبير (≥100 ريال)** → أحمر

## 🔐 قواعد الحفظ

1. **لا يمكن الإرسال** إذا:
   - ❌ يوجد عجز في الصندوق
   - ❌ لم يؤكد الكاشير البيانات

2. **يمكن الإرسال مع تحذير** إذا:
   - ⚠️ يوجد عجز في الأوزان (لكن يُسجل كتنبيه)

3. **بعد الإرسال:**
   - ✅ التقرير يُقفل
   - ✅ يصبح "رصيد أمس" للتقرير التالي
   - ✅ لا يمكن التعديل
