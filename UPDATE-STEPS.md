# تعليمات تحديث حقول الخطوات

## الخطوة 1: تنفيذ SQL

افتح Supabase SQL Editor وقم بتنفيذ الملف:
```
supabase/02-update-step-fields.sql
```

## الخطوة 2: إعادة تشغيل السيرفر

بعد تنفيذ SQL، أعد تشغيل السيرفر:
```bash
npm run dev
```

## ما تم تغييره:

### 1. Step 3 (المبيعات)
**قبل:**
- حاشي بالعظم (وزن + سعر)
- حاشي صافي (وزن + سعر)
- عجل بالعظم (وزن + سعر)
- عجل صافي (وزن + سعر)

**بعد:**
- حاشي (وزن + سعر) فقط
- عجل (وزن + سعر) فقط

### 2. Step 5 (المتبقي بالثلاجة)
**مضاف:**
- مخلفات الحاشي (كجم)
- مخلفات الغنم (كجم)
- مخلفات العجل (كجم)

### 3. حساب العجز
```
العجز = رصيد أمس + وارد اليوم - مبيعات - صادر - مخلفات
```
